Hướng dẫn sử dụng

1.Đầu tiên,các bạn phải kết nối Server, điền đầy đủ thông tin :

  • +Server Name
  • +Authemtication (Có 2 chế độ SQL Server Authentication OR Windows Authentication)
  • +Login ,Password(Nếu dùng chế độ SQL Server Authentication )
  • +Database Name

2.Sau khi kết nối thành công,bạn phải đăng nhập tài khoản đã được cấp phát.

3.Sau khi đăng nhập thành công màn hình chính xuất hiện, có 5 modul cơ bản của hệ thống :Trang chủ, Quản Trị Hệ Tống, Quản Lí,Giữ Liệu Thống Kê, Trợ Giúp, Giới Thiệu

4.Khi ấn vào Modul Quản Lí, chọn Đầu Sách,có 5 chức năng chính:

  • +Thêm Đầu Sách
  • +Sửa thông tin Đầu Sách
  • +Xóa Đầu Sách
  • +Lưu Đầu Sách
  • +Hủy Đầu Sách


4.Khi ấn vào Modul Quản Lí, chọn Độc Giả,có 6 chức năng chính:

  • +Thêm Độc Giả
  • +Sửa thông tin ĐỘc Giả
  • +Xóa Độc Giả
  • +Refresh(Cập nhật) ĐỘc Giả
  • +Lưu Độc Giả
  • +Hủy ĐỘc Giả


4.Khi ấn vào Modul Quản Lí, chọn Mượn Trả,có 6 chức năng chính:

  • +Xem
  • +Mượn Sách
  • +Trả Trả Sách
  • +Ghi Nhận
  • +Sửa
  • +Xóa
